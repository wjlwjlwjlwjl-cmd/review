* **空间复杂度**：空间复杂度指的是一个算法在不同的输入下，根据输入大小而变化的空间大小。比如我一个算法只是开了一个固定大小的数组，输入内容不会影响这个数组的大小，那么这个算法的空间复杂度就是 `O(1)`；而归并排序中，常使用的辅助数组，它的大小就会随输入待排序数组大小的改变而改变，因此归并排序的空间复杂度是 `O(N)`

* **顺序表与ArrayList**

    1. 线性表的线性指的是逻辑线性，而不是存储空间的，因此链表、循环队列、顺序表都是线性表；二叉树不是线性表

    2. 一般情况下，链表的插入删除比顺序表要高，但是当链表没有保存尾结点引用时，就必须遍历到尾结点，但是顺序表就不用

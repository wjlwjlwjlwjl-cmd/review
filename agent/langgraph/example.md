# 自创新运算：△运算（三角运算）
## 一、什么是△新运算？
我们平时学过加、减、乘、除四种基础运算，现在我们自己定义一种全新的简单运算，用符号 **△** 来表示。

规定新运算规则：
对于任意两个自然数 \(a\)、\(b\)，
\[
a△b = a \times 2 + b
\]

用大白话解释：
一个数△另一个数，等于**第一个数先乘2，再加上第二个数**。

举两个小例子帮你理解：
例1：\(3△4\)
第一步：第一个数字3×2=6
第二步：6+4=10
所以 \(3△4=10\)

例2：\(5△1\)
5×2+1=11，也就是 \(5△1=11\)

注意：△运算不可以随便交换前后数字，\(a△b\) 和 \(b△a\) 结果大多不一样。
比如 \(3△4=10\)，但 \(4△3=4\times2+3=11\)，两个结果不相等。

## 二、简单练习题
1. 按照上面定义的△新运算，计算：\(6△7\) 等于多少？
2. 若 \(x△3=15\)，请求出未知数 \(x\) 的值。

## 参考答案
1. \(6△7=6\times2+7=12+7=19\)
2. 由 \(2x+3=15\)，解得 \(x=6\)
